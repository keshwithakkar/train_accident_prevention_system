This project is an Arduino-based distance measurement system that uses an ultrasonic sensor (HC-SR04) to detect the distance of nearby objects. Based on the measured distance, multiple LEDs are activated in sequence to visually indicate how close or far the object is. As the object gets closer, more LEDs turn ON, providing a simple and effective distance indication system. The project uses Arduino (C++) and components such as an ultrasonic sensor and LEDs, making it useful for applications like obstacle detection, parking assistance, and distance monitoring.
